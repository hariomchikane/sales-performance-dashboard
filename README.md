# Sales Performance Dashboard 📊

## 📊 Project Overview
This project analyzes sales performance of executives using Microsoft Excel.

## 🎯 Objectives
- Track total sales of employees
- Compare target vs actual performance
- Identify top performers
- Analyze regional performance

## 🛠 Tools Used
- Microsoft Excel (Pivot Tables, Charts, Dashboard)

## 📈 Features
- Interactive dashboard
- Sales executive-wise analysis
- Target achievement tracking
- Region-based filtering
